# QA Call Assistant Demo

A simple Next.js + Tailwind demo for live QA call support.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repo into Vercel.
3. Deploy.

## Notes

- The tool stores nothing permanently.
- All state is in memory only.
- Refreshing the page clears the data.
